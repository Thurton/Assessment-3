How to Play:
Aaron the Alien is a Horizontal 2D scroller. You play as Aaron the Alien. The player can move around the screen on both x and y axis to the full length of  the screen height and width. Use the controls listed below to maneuver the player around the scrolling map and shoot enemies as they appear. Once you have progressed far enough though the level you will complete the game.

Controls:
W / Up Arrow Key = Up
A / Left Arrow Key = Left
S / Down Arrow Key = Down
D / Right Arrow Key = Right
Spacebar = Shoot
Enter = Play Game
